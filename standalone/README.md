# Standalone версия сайта "Защита Бизнеса"

## Описание

Полностью автономная статическая копия сайта без использования Vite, React или других сборщиков. Работает на чистом HTML, CSS и Vanilla JavaScript.

## Структура файлов

```
standalone/
├── index.html           # Главный HTML файл
├── css/
│   └── styles.css      # Дополнительные стили
├── js/
│   ├── app.js          # Основная логика приложения
│   └── admin.js        # Админ-панель и редактирование
├── data/
│   └── site-data.json  # Контент сайта
└── README.md           # Эта инструкция
```

## Возможности

### Основной функционал:
- Навигация по секциям с плавной прокруткой
- Адаптивное мобильное меню
- Динамическая загрузка контента из JSON
- Анимации и визуальные эффекты
- Изменение стиля хедера при скролле
- Подсветка активной секции в меню

### Админ-панель:
- Авторизация (пароль: `admin123`)
- Редактирование всего текстового контента
- Изменение логотипа (загрузка изображений)
- Автоматическое сохранение в localStorage
- Экспорт изменений в JSON файл

## Запуск

### Локально:

1. Откройте `index.html` в браузере напрямую, или
2. Используйте локальный веб-сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (http-server)
npx http-server

# PHP
php -S localhost:8000
```

Откройте в браузере: `http://localhost:8000`

### На хостинге:

Просто загрузите всю папку `standalone/` на любой веб-хостинг (GitHub Pages, Netlify, Vercel, обычный хостинг).

## Использование админ-панели

1. Откройте сайт в браузере
2. Нажмите кнопку "admin" в футере
3. Введите пароль: `admin123`
4. Наведите на любой элемент и начните редактирование
5. Изменения сохраняются автоматически в localStorage
6. Для сохранения на сервер:
   - Нажмите "Скачать изменения"
   - Замените файл `data/site-data.json`
   - Загрузите на хостинг

## Внешние зависимости

Используются только через CDN (не требуют установки):

- **Tailwind CSS** - стилизация
- **Lucide Icons** - иконки

Все работает без подключения к интернету после первой загрузки (браузер кэширует CDN).

## Технические особенности

- Без зависимостей от Node.js, npm, bundlers
- Кроссбраузерная совместимость
- Полностью responsive дизайн
- Оптимизированная производительность
- Простая структура для поддержки

## Настройка контента

Отредактируйте `data/site-data.json` для изменения контента:

```json
{
  "logo": "URL_логотипа",
  "hero": {
    "title": "Заголовок",
    "subtitle": "Подзаголовок"
  },
  "services": {
    "items": [...]
  },
  ...
}
```

## Смена пароля админ-панели

Откройте `js/admin.js` и измените:

```javascript
const ADMIN_PASSWORD = 'admin123';  // ваш новый пароль
```
